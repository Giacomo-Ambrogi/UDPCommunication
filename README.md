UDPCommunication

Progetto Java che realizza una comunicazione Client-Server tramite protocollo UDP.

- Il client invia un messaggio al server sulla porta 3000.
- Il server riceve il messaggio e lo rimanda indietro (echo).
- Il client riceve la risposta dal server.

Come eseguirlo

- Avvia MainServer
- Avvia MainClient

Cosa dimostra

- Uso delle socket UDP
- Invio e ricezione di datagrammi
- Comunicazione senza connessione (UDP)
